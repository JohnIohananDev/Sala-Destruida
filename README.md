# Sala-Destruida
Menu Para os Script do CMSP &amp; SALA DO FUTURO

Código desenvolvido para o menu que facilita a utilização dos scripts. Todos os direitos e créditos são reservados aos desenvolvedores da plataforma Destroyer.

Para utilizar, copie o conteúdo do bookmarklet e adicione-o aos seus favoritos do navegador. Em seguida, acesse o site desejado e clique no favorito para executá-lo.
